# Быстрый старт (5 минут)

## Шаг 1: Установка (1 минута)

```bash
# Клонируем репозиторий
git clone https://github.com/ВАШ_НИК/1c-repo-map.git
cd 1c-repo-map

# Устанавливаем зависимость (только tiktoken)
pip install -r requirements.txt
```

> **Windows-пользователям:** если `pip` не найден, используйте `python -m pip install -r requirements.txt`

## Шаг 2: Подготовка конфигурации (1 минута)

У вас должна быть **XML-выгрузка конфигурации 1С** (не EDT!).

Как получить:
1. Откройте конфигуратор 1С
2. Конфигурация → Выгрузить конфигурацию в файлы
3. Выберите папку, например `C:\Configs\MyConfig`

Структура должна быть такой:
```
MyConfig/
├── Configuration.xml
├── ConfigDumpInfo.xml
├── Catalogs/
├── Documents/
├── CommonModules/
├── ...
```

## Шаг 3: Генерация карт (2-3 минуты)

```bash
python src/generate_repo_map.py \
  --config "C:\Configs\MyConfig" \
  --output "./repo_map"
```

## Шаг 4: Использование с LLM (1 минута)

Откройте `repo_map/README.md` — это точка входа.

### Пример запроса в ChatGPT / Claude

Скопируйте в промпт:

```
Я работаю с конфигурацией 1С. Вот репозиторная карта:

=== README.md ===
[вставьте содержимое README.md]

=== Documents.md (фрагмент) ===
[вставьте раздел с нужным документом]

Задача: [опишите задачу]

Конкретные файлы:
- [путь к .bsl файлу]
```

**Правило:** Карты идут ПЕРЕД исходными файлами — LLM сначала получает навигацию, потом код.

## Шаг 5: Тестирование (опционально)

```bash
python src/test_tokens.py
```

Скрипт покажет, сколько токенов экономит использование карт на этапе навигации по конфигурации (поиск объектов, форм и модулей).

## Типичные проблемы

### «ConfigDumpInfo.xml не найден»

Вы указали не ту папку. Убедитесь, что в папке есть `Configuration.xml` и `ConfigDumpInfo.xml`.

### «Парсинг подсистем... Корневых подсистем: 0»

Это нормально для некоторых форматов выгрузки. Карты всё равно сгенерируются, просто без иерархии подсистем.

### Ошибка кодировки на Windows

Если видите кракозябры в консоли, запускайте через PowerShell с UTF-8:
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
$env:PYTHONIOENCODING="utf-8"
python src/generate_repo_map.py --config "..." --output "..."
```

### Большой размер objects_index.md

Это нормально — индекс всех 12 000 объектов. При работе с LLM не вставляйте его целиком, если задача узкая. Используйте поиск по ключевым словам.
